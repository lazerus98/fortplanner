Basics
Left-click paints the active tool, right-click erases. Drag to paint across multiple cells. Grid size, zoom, and Z-level are controlled in the top bar.

Shapes & brush
Use the Shape dropdown (Freehand / Rect / Line / Circle / Fill) plus the Brush size to control how you paint. "Hollow" draws only outlines for Rect/Circle.

Mirror & axes
Mirror V / H paints symmetrically across the green axes. Drag the axis handles in the grid to move the mirror lines.

Z-levels & dig overlay
Z+ / Z− switch floors and extend when you go past the top/bottom. Scrolling over the grid only navigates within existing Z-levels (no auto-add). The "Preview" button shows the next Z-level as a faded overlay to help align towers, stair stacks, etc. Dig designations from the #dig tab are shown as a green overlay on the other tabs for alignment.

Selection, paste & transforms
"Select Area" → drag to mark a rectangle. Use Copy / Cut / Paste / Clear in the status bar. In Paste mode you get a ghost preview under the cursor; right-click or Esc cancels. "Skip blanks" avoids overwriting non-empty cells. Rotate / Mirror X / Mirror Y transform the paste buffer before placing.

CSV / Quickfort I/O
"Copy to clipboard" copies the current tab as multi-Z quickfort CSV. "Download CSV" saves the current tab, "Export All" bundles all tabs into one file, "Load" imports quickfort CSV (multi-tab, multi-Z supported).

Undo & keyboard shortcuts
Undo / Redo step through your changes.
Ctrl/Cmd+Z / Ctrl/Cmd+Y (or Ctrl/Cmd+Shift+Z) – undo/redo
Ctrl/Cmd + + / − / 0 – zoom in/out/reset
Ctrl/Cmd+C – copy current selection
Ctrl/Cmd+V – toggle paste mode
Esc – clear selection or exit paste mode
